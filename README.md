####To Run The Game File####

//////////////////////////


####Main Controls####

* Press 'E' to open inventory
* Press 'P' to toggle particle effect
* W,A,S,D to movement
* Arrow keys to movement
* Press 'F5' to restart game
* Press 'F11' for toggle fullscreen mode
